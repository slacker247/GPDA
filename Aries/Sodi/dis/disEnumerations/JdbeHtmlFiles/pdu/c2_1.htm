Beam Parameter Index Field

Size: 16
Data Type: Unsigned
Description: This field shall specify a beam parameter index number that shall be used by receiving entities in conjunction with the emitter name field to provide a pointer to the stored database parameters required to regenerate the beam. This field shall be represented by a 16-bit unsigned integer.
