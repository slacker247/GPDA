Dead Reckoning Parameter Record

Size: 320
Description: Used to provide the parameters for dead reckoning the position and orientation of the entity. Dead Reckoning Algorithm in use, Entity Acceleration and Angular velcoity shall be included as a part of the dead reckoning parameters. 120 bits are reserved for other parameters that are currently undefined.

Record Components:

Item Name Bit Length Opt Opt Ctl Rpt Rpt Ctl
Dead Reckoning Algorithm Field 8
Dead Reckoning Other Parameters Field 120
Entity Linear Acceleration Record 96
Entity Angular Velocity Record 96
