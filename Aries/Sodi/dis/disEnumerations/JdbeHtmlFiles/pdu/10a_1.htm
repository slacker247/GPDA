Antenna Pattern Parameter Record Varient

Size: 312
Description: The Antenna Pattern Parameter Record shall be represented by one of the following records. Which record is used is determined by the value of the Antenna Pattern Type field. Omni-direction and beam antenna pattern types shall be supported by all receiver simulation appliactions. Support for other antenna pattern types is optional.
