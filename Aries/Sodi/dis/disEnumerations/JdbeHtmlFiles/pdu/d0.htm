High Density Track/Jam Field

Size: 8
Data Type: Enumerated
Description: This field shall be used to indicate whether or not the receiving simulation application can assume that all targets, in the scan pattern which the sending emitter can track (for a phased array system) or jam (for a jamming system), are being tracked or jammed respectively. When high density track/jam is selected, the system issuing the PDU shall not include any entries in the track/jam field. This field shall be represented by an 8-bit enumeration (see Section 8 in EBV-DOC).

Enumerations:

Value Description
0 Not Selected
1 Selected
