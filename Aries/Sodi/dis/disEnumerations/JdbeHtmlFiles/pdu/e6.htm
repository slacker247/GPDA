Input Source field

Size: 8
Data Type: enumerated
Description: This field shall specify which position (pilot, co-pilot, first officer, gunnery officer, etc.) or data port in the entity utilizing the radio is providing the input audio or data being transmitted. This field shall be represented by an 8-bit enumeration. The values of this enumeration shall be independently specified for each Entity Type (see Section 9 in EBV-DOC).

Enumerations:

Value Description
0 Other
1 Pilot
2 Copilot
3 First Officer
4 Driver
5 Loader
6 Gunner
7 Commander
8 Digital Data Device
9 Intercom
