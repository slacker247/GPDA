Antenna Pattern Type field

Size: 16
Data Type: enumerated
Description: This field shall specify the type of representation used for the radiation pattern from the antenna. The value of this field shall determine the interpretation of the Antenna Pattern Parameter field. This field shall be represented by a 16-bit enumeration (see Section 9 in EBV-DOC).

Enumerations:

Value Description
0 Omni-directional
1 Beam
2 Spherical harmonic
