Viewing Back-References

In a complex project, it is possible to have many different relationships between classes, instances, and slots. The Back-References Pane allows you to view a list of all the frames that reference a highlighted item.

To view the Back-References Pane:

  1. Highlight the item for which you wish to see a list of back references. You must make sure to go to the correct location:
    For a class, go to the Class Tab and highlight the item in the Class Relationship Pane.
    For a slot, go to the Slots Tab and highlight the item in the Slots Pane. 
    For an instance, go to the Instances Tab, select the class where the instance appears in the Class Pane at the Instances Tab, then highlight the item in the Direct Instances Pane.
  2. Click on the Back-References button at the top right of the pane.

For classes, it is also possible to view those classes that are related through a particular slot. See Viewing Class Relationships for more information.
