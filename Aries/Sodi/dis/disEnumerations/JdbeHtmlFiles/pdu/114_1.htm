Beam Antenna Pattern Record Record

Size: 256
Description: The Antenna Pattern Parameters field shall be Beam Antenna Pattern when the Antenna Pattern Type field has a value of 1. This record specifies the direction, pattern, and polarization of radiation from a radio transmitter's antenna. The radiation pattern produced by the antenna is defined with respect to a Beam Coordinate system. The Beam Coordinate system is a right handed Cartesian coordinate system with the center of the beam along the x axis of the system (See Figure 20). In the Transmitter PDU, the origin of the beam coordinate system is specified by the Antenna Location record. The Beam Antenna Pattern Record contains the following fields:
