Making a Metaclass From an Existing Class

You can turn any existing class in your project into a metaclass. This allows you to take any class you have developed and use its attributes for classes that are unrelated in the hierarchy. You can make a class into a metaclass in one of two ways:

  • By adding a metaclass as an additional superclass of the class
  • By changing the superclass of the class to a metaclass

Replacing a Superclass with a Metaclass

Unless a class already has multiple superclasses, changing its metaclass will remove it and all of its subclasses from the standard hierarchy.  Its instances will not be visible in the Instance Tab. To change the superclass of a class to a metaclass:

  1. Click on the class in the Class Relationship pane. 
  2. Drag the class to the desired metaclass. The selected superclass must itself be a metaclass, as indicated by a green class icon . 
  3. The superclass of the dragged class will change to the selected metaclass. The dragged class, and all of its subclasses, will inherit the properties of the selected metaclass.
