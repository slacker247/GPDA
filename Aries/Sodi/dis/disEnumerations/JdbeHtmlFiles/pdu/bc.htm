Emitter System Record

Size: 32
Description: Information about a particular emitter shall be rpresented using an Emitter System Record. This record shall consist of three fields: Emitter Name, Function and Emitter ID Number. These fields are described Below:

a) Emitter Name - This field shall specify the emitter name for a particular emitter. This field shall be represented by a 16-bit enumeration (see Section 8 in Document EBV-DOC).
b) Function - This field shall specify the function for a particular emitter. Typical functions include airborne fire control, ground surveillance radar, etc. This field is intended to help receiving entities determine if the Electromagnetic Emission PDU is of interest to the systems simulated by that entity. This field shall be represented by an 8-bit enumeration (see Section 8 in EBV-DOC).
c) Emitter Identification Number - This field shall specify the emitter identification number for a specific emitter system. This field shall be represented by an 8-bit unsigned integer. This field allows the differentiation of multiple emitter systems on an entity even if in some instances two or more of the systmes may be identical emitter types. Once established for an exercise, the emitter ID for each system shall not be changed during that exercise. The Emitter System Record is represented in Table 14.
