State Update Indicator Field

Size: 8
Data Type: Enumerated
Description: This field shall be used to indicate if the data in the PDU represents a state update or just data that has changed since issuance of the last Emitter PDU (relative to the identified entity and emission system(s)). This field shall be represented by an 8-bit enumeration. (see Section 8 in EBV-DOC).

Enumerations:

Value Description
0 State Update
1 Changed Data Update
