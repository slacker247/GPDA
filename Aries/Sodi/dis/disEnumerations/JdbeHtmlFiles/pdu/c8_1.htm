Pulse Width Field

Size: 32
Data Type: Float
Description: This field shall specify the average pulse width of the emission in micro-seconds. Pulse modulation for a particular emitter and mode shall be derived from database parameters stored in the receiving entity. This field shall be represented by a 32-bit floating point number.
