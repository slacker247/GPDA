Displaying the Classes & Instances Tab

To display the Classes & Instances tab:

  1. Select Configure... from the Project menu.
    The Configure dialog box opens.
  2. Select ClsesAndInstancesTab by clicking on it.
  3. Click OK.
    The Protégé window now has an additional tab.
  4. Click on the Classes & Instances tab to view a single interface where you can edit both classes and instances.
