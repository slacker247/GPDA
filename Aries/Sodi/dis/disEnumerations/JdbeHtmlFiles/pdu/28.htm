Entity Type Record

Size: 64
Description: The type of entity in a DIS exercise shall be specified by an Entity Type record. This record shall specify the kind of entity, the country of design, the domain, the specific identification of the entity, and any extra information necessary for describing the entity. Fields not used shall contain the value zero.
