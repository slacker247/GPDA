Entity Domain Field

Size: 8
Data Type: Enumerated
Description: This field shall specify the domain in which the equipment operates (for example, subsurface, surface, land, etc.) except for munition entities. For Munition entities this field shall specify the domain of the target (for example the munition might be surface-to-air, so the domain would be anti-air).
NOTE: This enumeration will not be stored in the enumeration table, but rather is a part of the Entity Type Database.
