Entity Identity Field

Size: 16
Data Type: Unsigned
Description: Each entity in a given DIS application shall be given an Entity Identifier (Entity ID) unique to all other entities in that application and in that exercise. This identifier is valid for the duration of the exercise; however, entity IDs shall be reused when all possible entity IDs have been exhausted. No entity shall have an ID of zero, (2e16 -1) or (2e16 -2). This number need not be registered or retained for future exercises.
