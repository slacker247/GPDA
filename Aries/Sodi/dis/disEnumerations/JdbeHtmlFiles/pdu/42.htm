Entity Trailing Effect Field

Size: 2
Data Type: Enumerated
Description: Describes the size of trailing effects from an entity. Trailing effects include dust clouds for land platforms, wakes for surface platforms, rocket flames for guided munitions, contrails and ionization trails for air paltforms, etc.

Enumerations:

Value Description
0 None
1 Small
2 Medium
3 Large
