Importing RDF Files

You can convert a project in RDF format to Protégé-2000 by importing it. The original project is not altered. If you have no schema for your RDF (your instances do not have a declared type) the system will create a single class with the required properties and all imported instances will be instances of this class. You can then use Protégé-2000 to create a schema and to assign your existing instances to the appropriate classes.

To import existing RDF files:

  1. Select Import Project... from the Project menu. 
  2. Select Resource Description Framework (RDF) from the Select Format dialog box and click OK.
  3. A Resource Description Framework (RDF) dialog box is displayed. 
  4. Enter the name of the file containing your RDF schema (if any) in the Classes file name field. To browse for a file, click the button.
  5. Enter the name of the file containing your RDF instance data in the Instances file name field.  To browse for a file, click the button.
  6. Enter your namespace in the Namespace field.

  7. Press the OK button and the system will load the schema and instances. 

For more information about RDF files and Protégé, see RDF Support in Protégé-2000.
