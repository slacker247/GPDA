The DIS Data Dictionary contains the message structures and enumeration values for the DIS Application Protocol in a database format. The database was created by Joint Data Base Elements (JDBE) at Ft. Huachuca, Arizona. The dictionary can be viewed either on the web by selecting the links below or you can download a copy of the Microsoft Access database from the ftp site. For questions regarding the content of the DIS Data Dictionary, please contact Jeff Wicks, jwicks@ist.ucf.edu.
